Backend Overview

An overview of the backend classes of the Lighting GUI.

The backend handles communicating with the lights, storing data about the session, and generating the icons for the GUI.

CommLayer

The CommLayer class provides communication protocols that allow the user to connect and send packets to an LED array. Currently it supports serial, UDP, and HTTP.

DataLayer

The DataLayer class stores and maintains the data about the state and settings of the application. It also saves the settings sent to the LED hardware, such as the current brightness and current lighting routine.

IconData

The IconData class creates all the pixel-y square icons that are seen throughout the app. The grids are made by using a very small buffer of RGB values to do the computation. For display, this buffer gets upscaled to the desired icon size.
