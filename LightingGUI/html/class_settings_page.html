Detailed Description

It provides the ability to change the speed the LEDs update and the amount of minutes it takes for the LEDs to timeout.

It also provides a way to switch between different communication types such as Serial and UDP. This interface automatically populates serial connections, but allows users to add and remove connections for UDP and HTTP.
